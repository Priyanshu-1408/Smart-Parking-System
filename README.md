# Smart Parking Lot System

## Objective
To develop a smart parking lot application that automatically allocates parking slots based on vehicle requirements.

## Features
- Add parking slots
- View all parking slots
- Park vehicles based on EV and Covered requirements
- Automatically allocate nearest available slot
- Remove vehicles and free slots
- Display output messages

## Parking Allocation Logic
1. Filter unoccupied slots
2. Match EV and Covered requirements
3. Sort by slot number
4. Allocate nearest slot
5. Show message if no slot available

## Tech Stack
- React
- JavaScript
- Vite
